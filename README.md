# 🪙 CoinCounter – AI-Powered Coin Detection & Analysis App

**CoinCounter** is an intelligent Android application that utilizes computer vision and machine learning to detect, classify, and analyze coins in real time. Powered by a custom-trained dataset from Roboflow, the app provides accurate coin recognition and automated value computation to assist users in counting and decision-making.

---

## 📌 Overview

The **CoinCounter App** leverages object detection models trained on a Roboflow dataset to identify different coin types through the device camera. It processes visual data, detects objects, and computes the total monetary value based on recognized coins.

This project demonstrates the integration of AI/ML into mobile applications, combining real-time image processing with practical financial utility.

---

## ✨ Features

### 📷 Real-Time Coin Detection

* Detect coins using the device camera
* Identify multiple coins simultaneously

### 🧠 AI-Based Object Recognition

* Trained using a custom dataset from Roboflow
* Accurate classification of different coin denominations

### 🪙 Automatic Coin Counting

* Counts detected coins instantly
* Handles multiple objects in a single frame

### 💰 Value Calculation

* Computes total monetary value based on detected coins
* Supports configurable coin denominations

### 🖼️ Image Processing

* Analyze captured images for coin detection
* Bounding boxes and labels for detected objects *(if implemented)*

## 🤖 How It Works

1. Capture or stream an image using the camera
2. Run the trained object detection model
3. Identify and classify each coin
4. Count detected coins
5. Calculate total value based on predefined denominations
